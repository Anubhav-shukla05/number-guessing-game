## Game Logic

1. The computer generates a random number between 1 and 50.
2. The player chooses a difficulty level.
3. The player guesses the number.
4. The game gives hints:
   - Too High
   - Too Low
5. The game ends when:
   - The player guesses correctly
   - Attempts become zero
